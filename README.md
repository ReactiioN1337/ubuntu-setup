## How to use

```sh
wget --no-check-certificate \
https://raw.githubusercontent.com/frk1/ubuntu-setup/master/ubuntu-setup.sh \
&& chmod +x ubuntu-setup.sh \
&& ./ubuntu-setup.sh
```

Single line:

```sh
wget --no-check-certificate https://raw.githubusercontent.com/frk1/ubuntu-setup/master/ubuntu-setup.sh && chmod +x ubuntu-setup.sh && ./ubuntu-setup.sh
```
